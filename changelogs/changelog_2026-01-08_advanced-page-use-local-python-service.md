# Changelog — 2026-01-08 — advanced-page-use-local-python-service

## Було
- Розширений режим ([suno-clone/src/pages/AdvancedPage.tsx](../suno-clone/src/pages/AdvancedPage.tsx)) викликав Supabase Edge Function `generate-music`.
- Edge Function намагалася форвардити запит у `http://127.0.0.1:8000`, але це середовище “хмари”, тому локальний сервіс недоступний.
- У бібліотеці треки могли залишатися без коректного `audio_url`, що призводило до помилки «Помилка завантаження аудіо» під час відтворення.

## Стало
- Розширений режим напряму викликає локальний `python-service` за адресою `http://localhost:8000/generate-music` (аналогічно сторінці Create).
- Генерація коректно створює трек, оновлює `audio_url` після завершення й відтворення в бібліотеці працює стабільно.

## Ефект
- Прибрано залежність від неможливого виклику `localhost` з Edge Function.
- Зменшено кількість випадків “битих” треків без аудіо та помилок відтворення.
