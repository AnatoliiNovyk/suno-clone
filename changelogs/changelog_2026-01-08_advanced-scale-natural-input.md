# Changelog — 2026-01-08 — advanced-scale-natural-input

## Було
- Поле Scale у розширеному режимі очікувало, що користувач введе саме enum значення Lyria (наприклад `G_MAJOR_E_MINOR`).
- Введення «E minor» / «G major» не конвертувалось автоматично, що ускладнювало використання.

## Стало
- Додано автоматичну конвертацію введення на кшталт «E minor» / «G major» (також підтримано «E мінор» / «G мажор») у відповідний enum Lyria Scale.
- У UI показується підказка, у що саме буде конвертовано введене значення.

## Ефект
- Менше помилок введення і швидше налаштування тональності.
- Зручніше задавати тональність у звичному для музикантів форматі.
