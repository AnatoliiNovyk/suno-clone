# Changelog: Fix Audio Format (WAV)

## Дата зміни / Date of Change
2026-01-08

## Короткий опис / Short Description
fix-audio-format-wav-header

## Опис змін / Description of Changes

### До зміни / Before Change
- Сервіс отримував "сирі" PCM байти від Google Gemini API.
- Зберігав їх з розширенням `.mp3`, але без заголовків.
- Браузери не могли відтворити файл або показували тривалість 0:00.

### Після зміни / After Change
- Додано імпорт модуля `struct`.
- Реалізовано функцію `create_wav_header`, яка генерує стандартний 44-байтний заголовок для WAV.
- Перед завантаженням в Storage, до PCM даних додається WAV заголовок.
- Змінено розширення файлу на `.wav` та MIME-тип на `audio/wav`.
- Встановлено частоту дискретизації 24000 Гц (стандарт для Gemini 2.0 Flash Exp).

### Переваги / Benefits
- Коректне відтворення згенерованої музики в браузері.
- Виправлено проблему нульової тривалості треку.
